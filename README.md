# PriceTracker 📊

منصة متعددة المستخدمين لمراقبة أسعار المنتجات مع إشعارات تليجرام فورية.

## المميزات ✨

- ✅ تسجيل ودخول المستخدمين مع تشفير كلمات المرور
- ✅ ربط تليجرام لتأكيد الهوية وإرسال الإشعارات
- ✅ فحص أسعار تلقائي كل 5 دقائق
- ✅ دعم منصات متعددة (EasyOrders, WooCommerce, Shopify)
- ✅ واجهة عربية جميلة وسريعة
- ✅ فصل بين "منتجات متجري" و "مراقبة المنافسين"
- ✅ سجل تاريخي كامل للأسعار
- ✅ إشعارات فورية عند تغير الأسعار

## المتطلبات 📋

- Python 3.8 أو أحدث
- حساب على Railway (للنشر)
- بوت تليجرام (من BotFather)

## التثبيت المحلي 💻

### 1. استنساخ المشروع

```bash
git clone https://github.com/your-username/PriceTracker.git
cd PriceTracker
```

### 2. إنشاء بيئة افتراضية

```bash
python -m venv venv
source venv/bin/activate  # على Linux/Mac
venv\Scripts\activate  # على Windows
```

### 3. تثبيت المكتبات

```bash
pip install -r requirements.txt
```

### 4. إعداد المتغيرات البيئية

```bash
cp .env.example .env
```

عدل ملف `.env` وأضف:
- `SECRET_KEY`: مفتاح سري عشوائي
- `TELEGRAM_BOT_TOKEN`: توكن بوت تليجرام
- `TELEGRAM_WEBHOOK_URL`: رابط موقعك (محلياً: http://localhost:5000)
- `PRICE_CHECK_INTERVAL`: فترة الفحص بالدقائق (افتراضي: 5)

### 5. تشغيل التطبيق

```bash
python app.py
```

افتح المتصفح على: `http://localhost:5000`

## إنشاء بوت تليجرام 🤖

### 1. التحدث مع BotFather

1. افتح تليجرام وابحث عن `@BotFather`
2. ابدأ المحادثة واكتب `/newbot`
3. اتبع التعليمات لإنشاء البوت
4. احصل على التوكن (Token) واحفظه

### 2. إعداد Webhook

بعد نشر التطبيق على Railway، سجل الـ webhook:

```bash
curl -F "url=https://your-app-url.railway.app/telegram/webhook" \
     https://api.telegram.org/botYOUR_BOT_TOKEN/setWebhook
```

## النشر على Railway 🚀

### 1. إنشاء حساب على Railway

سجل في [railway.app](https://railway.app)

### 2. إنشاء مشروع جديد

1. اضغط على "New Project"
2. اختر "Deploy from GitHub repo"
3. اختر مستودع المشروع الخاص بك

### 3. إضافة المتغيرات البيئية

في إعدادات المشروع (Variables)، أضف:

```
SECRET_KEY=your-random-secret-key-here
DATABASE_URL=sqlite:////data/pricetracker.db
TELEGRAM_BOT_TOKEN=your-telegram-bot-token
TELEGRAM_WEBHOOK_URL=https://your-app-url.railway.app
PRICE_CHECK_INTERVAL=5
PORT=5000
```

### 4. إعداد قاعدة البيانات المستمرة

لضمان بقاء قاعدة البيانات بعد إعادة النشر:

1. في إعدادات المشروع، أضف Volume:
   - Mount path: `/data`
   
2. عدل `DATABASE_URL` إلى:
   ```
   DATABASE_URL=sqlite:////data/pricetracker.db
   ```

### 5. نشر المشروع

اضغط على "Deploy" وانتظر حتى يكتمل النشر.

### 6. تسجيل Webhook

بعد النشر، سجل webhook البوت:

```bash
curl -F "url=https://your-app-url.railway.app/telegram/webhook" \
     https://api.telegram.org/botYOUR_BOT_TOKEN/setWebhook
```

### 7. اختبار البوت

أرسل `/start` إلى بوتك للتأكد من أنه يعمل.

## استخدام التطبيق 🎯

### 1. إنشاء حساب

1. افتح الموقع
2. اضغط على "إنشاء حساب جديد"
3. أدخل بياناتك
4. سيتم إنشاء حساب معلق

### 2. تفعيل الحساب

1. في صفحة "حسابك معلق"، اضغط على "تفعيل الحساب عبر تليجرام"
2. سيفتح بوت تليجرام
3. اضغط Start
4. شارك رقم هاتفك
5. عد إلى الموقع وسيتم تفعيل الحساب تلقائياً

### 3. إضافة روابط

1. في لوحة التحكم، أضف رابط منتجك في قسم "منتجات متجري"
2. أضف روابط المنافسين في قسم "مراقبة المنافسين"
3. سيبدأ النظام في فحص الأسعار تلقائياً

### 4. مراقبة الأسعار

- راقب آخر الأسعار في لوحة التحكم
- اضغط على أيقونة التاريخ لعرض سجل الأسعار الكامل
- ستصلك إشعارات تليجرام عند تغير الأسعار

## المميزات الإضافية الذكية 🧠

### 1. فلترة الإشعارات حسب نسبة التغيير

يمكنك تعديل الحد الأدنى لنسبة التغيير لإرسال الإشعارات (في الكود).

### 2. سجل تاريخي شامل

عرض تاريخ الأسعار الكامل لكل منتج مع رسوم بيانية مستقبلية.

### 3. مقارنة سريعة

إمكانية مقارنة أسعار منتجات متعددة في صفحة واحدة.

## استكشاف الأخطاء 🔧

### البوت لا يرد

1. تأكد من تسجيل webhook بشكل صحيح
2. تحقق من التوكن في المتغيرات البيئية
3. راجع سجلات Railway للبحث عن أخطاء

### قاعدة البيانات تُفرغ

تأكد من إعداد Volume في Railway والمسار الصحيح لقاعدة البيانات.

### الأسعار لا تُفحص

1. تحقق من أن الروابط صحيحة
2. راجع سجلات النظام لمعرفة الأخطاء
3. تأكد من أن Scheduler يعمل
